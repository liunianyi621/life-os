# LifeOS

项目名称：
LifeOS

定位：
个人生活操作系统

核心模块：

- 今日任务
- 坏习惯追踪
- 金币系统
- 奖励系统
- 连续天数
- 统计热力图

## 项目目标

做一个移动端优先的个人自律系统，用任务、金币、坏习惯惩罚、奖励兑换和统计反馈，帮助用户把日常自律行为游戏化。

## 当前状态

- 当前是纯静态 App，可直接打开 `index.html` 使用。
- 主入口是 `index.html`，样式在 `css/styles.css`，交互脚本在 `js/` 模块中。
- 已包含 PWA manifest、图标和 Vercel 静态部署配置。
- `outputs/life-rpg` 是生成产物，不再手动维护。
- 数据主要保存在浏览器本地。
- 已迁移到 `~/codex-workspace/Active/LifeOS`，作为长期维护项目继续管理。
- 已初始化本地 Git 仓库，但尚未创建远程仓库，也未推送到 Github。
- 暂未部署到 Vercel。
- 目前不重构，不改 React，不改 Vite，先保住可运行版本。

## 主要功能

- 今日任务管理
- 坏习惯记录与金币惩罚
- 笔记
- 奖励兑换
- 金币余额与连续天数统计
- 自律热力图和坏习惯趋势

## 下一步计划

- 下一步优先部署上线，先验证实际使用价值。
- 上线前不做工程化重构。
- 等功能稳定、方向明确后，再考虑是否重建为 React/Vite 或其他标准前端工程。

## 本地运行

直接在浏览器打开：

```text
index.html
```

## 构建预览副本

生成 `outputs/life-rpg`：

```text
node scripts/build-output.js
```

如果本机安装了 npm，也可以运行：

```text
npm run build
```
